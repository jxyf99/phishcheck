Privacy Policy

PhishCheck

Last updated: May 6, 2026

Overview

PhishCheck provides a rule-based phishing risk score for URLs and pasted messages. The Chrome extension sends the current tab URL to the PhishCheck backend so the service can return a score, status, and reasons.

Data We Process

When you use the extension, PhishCheck processes the current website URL you choose to scan. When you use the web app, PhishCheck processes the text or URL you paste into the analyzer.

How Data Is Used

The submitted URL or text is used only to generate the phishing risk result shown to you. PhishCheck does not sell this data and does not use it for advertising.

Storage

This MVP does not include user accounts or a database for storing scan history. Hosting infrastructure may process routine request metadata needed to operate and secure the service.

Sharing

PhishCheck shares requests with the hosting provider that runs the backend. PhishCheck does not share submitted URLs or pasted content with data brokers or advertising networks.

Security Note

PhishCheck is a rule-based educational and triage tool. It is not a guarantee that a website or message is safe.

Contact

For privacy questions, use the contact email listed on the Chrome Web Store listing for this extension.
